    設$f$爲實函數。\begin{enumerate}
        \item 若$f$可寫作$O+E$兩部分，其中$O$為奇函數，$E$為偶函數，試以$f$表述$O$與$E$。
        \item 考慮$(O+E)^n$。\begin{enumerate}
            \item 求$(f)^n$的偶函數部分。
            \item 求$(f)^n$的奇函數部分。
        \end{enumerate}
    \end{enumerate}

    證明當$x>-1$時，對於任意整數$n>0$，$$(1+x)^n\leq 1+nx$$
